# Health Monitoring Data Analysis

## Project Overview
This project analyzes health monitoring data to study blood pressure trends across different age groups and evaluate the relationship between smoking status and blood pressure using statistical analysis techniques.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Objectives
- Analyze blood pressure distribution across age groups
- Detect outliers using box plots
- Evaluate correlation between smoking status and blood pressure
- Generate healthcare-related insights from data

---

## Data Preprocessing
Performed data cleaning and preprocessing tasks:

- Missing value treatment
- Duplicate removal
- Data formatting
- Statistical preprocessing

---

## Statistical Analysis
### Point-Biserial Correlation
Used Point-Biserial Correlation to analyze the relationship between:

- Smoking Status (Binary Variable)
- Blood Pressure (Continuous Variable)

### Distribution Analysis
- Mean
- Median
- Standard Deviation
- Outlier Detection

---

## Visualizations
Generated visualizations including:

- Grouped Box Plots
- Distribution Charts
- Correlation Analysis Graphs

---

## Key Insights
- Identified blood pressure trends across age groups
- Detected abnormal blood pressure outliers
- Evaluated the impact of smoking on blood pressure
- Interpreted statistical relationships using correlation analysis
